Game Cellar is a MEAN stack e commerce web application that provides users with a platform where they can sell, trade, and keep wishlists of retro video games.
There is built in user authentication, as well as a profile page where users can display personal information for their peers to view. Profile also includes fully functional edit section so users can update their personal information at any time.
Future installments will include: game bidding, message board for users to communicate publicly and privately, and game veryfier.
